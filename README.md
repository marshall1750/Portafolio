### 🌐 2. Proyecto: Red de Campus Universitario

Diseño e implementación de una red segmentada para un campus universitario, aplicando buenas prácticas de networking.

#### 🔧 Tecnologías y conceptos:
- Subnetting
- VLANs
- Inter-VLAN Routing
- Switching capa 2 y capa 3
- Wireless LAN Controller (WLC)
- Redes WiFi (Estudiantes, Invitados)

#### 📌 Descripción:
Se diseñó una arquitectura de red segmentada por facultades, permitiendo una gestión eficiente del tráfico y mejorando la seguridad mediante segmentación lógica.

#### ⚙️ Implementaciones:
- Configuración de switches capa 2 y capa 3
- Creación de VLANs por facultad
- Enrutamiento inter-VLAN
- Integración de red inalámbrica mediante WLC
- Conectividad completa entre segmentos
- Asignación de IPs mediante DHCP

#### 🎯 Objetivo:
Garantizar conectividad eficiente, escalabilidad y segmentación adecuada en un entorno universitario.
